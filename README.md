
# Initialize
```
chmod +x avolut-backup
./avolut-backup 
```
it will create `backup.yaml` at first. edit it to backup files or db.

backup now: 
```
./avolut-backup 
```



# Service

install as service
```
./avolut-backup --service install
```

remove service
```
./avolut-backup --service remove
```
